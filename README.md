# 🚀 Blog Source (my-blog)

这是一个基于 [Hexo](https://hexo.io/) 框架与 [NexT](https://theme-next.js.org/) 主题搭建的个人博客源码仓库。采用了 **“源码与静态网页分离”** 的架构，通过本地进行构建，并自动部署至 GitHub Pages。

**部署后项目：** <https://github.com/StarHeartY/blog-build>

**个人网址：** [startyi.com](https://startyi.com)

## 🏗 架构说明

* **框架**: Hexo 7.x
* **主题**: NexT 8.x (Gemini Scheme)
* **源码仓库 (本仓库)**: `my-blog` - 存放 Markdown 原稿及主题配置文件。
* **部署仓库**: `blog-build` - 仅存放生成的静态文件，由 Hexo 自动推送。
* **域名**: [startyi.com](https://startyi.com) (Cloudflare 解析)。

## 🛠 开发环境

* **Node.js**: v20+
* **包管理器**: NPM
* **渲染引擎**: `hexo-renderer-kramed` (替代默认引擎以支持复杂数学公式)
